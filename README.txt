	这是一个基于STM32的智能轮椅系统


	设计分为两个部分，轮椅嵌入式硬件的的实现和云端服务器的实现。
	硬件部分使用两块主控芯片，分别负责轮椅电机驱动部分、信息采集和云端通讯部分。
	轮椅可实现GPS定位，GPRS无线通信，人体信息采集（心率，血氧），环境信息采集（温湿度）等功能（可随意愿扩充），此类信息可通过sim900传至云端服务器。
	轮椅实现的电机驱动，51为主控芯片。可遥控，使用WiFi传输控制信号（WiFi手动组网）。
	
	云端服务器通过监听指定端口接收sim900的http连接，从url中提取所需信息，
保存至数据库。完成轮椅系统数据的更新。
	服务器可接受来自浏览器的访问请求，验证身份（还没写，但比较简单），从数据库中读取对应信息，生成HTML网页，返回给用户，用户可借此获得gps、使用者生理信息，环境信息。
	
	服务端接入了百度地图的接口，所以返回的信息可以直接在地图上定位。
